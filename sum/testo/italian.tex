% % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % % %


\InputFile
Su standard input (\texttt{scanf}) sono presenti due righe, contenenti:
\begin{itemize}[nolistsep,itemsep=2mm]
\item Riga $1$: l'intero $N$, il numero di numeri da sommare
\item Riga $2$: gli $N$ interi $a_i$ da sommare, separati da spazi.
\end{itemize}

\OutputFile
Su standard output (\texttt{printf}) deve essere scritta esattamente una riga,
contenente un solo numero, ovvero la somma degli elementi dell'array.

\Constraints

\begin{itemize}[nolistsep, itemsep=2mm]
    \item $1 \leq N \leq 1\,000\,000 $.
    \item $-1\,000 \leq a_i \leq 1\,000$ per ogni $i$.
\end{itemize}
